📐 ผังงานคณิตศาสตร์ (Mathematical Flowchart)

ทำความเข้าใจการทำงานของอัลกอริทึมจากข้อความสู่เวกเตอร์และการจัดกลุ่ม

1. สมการความถี่ร่วม (Co-occurrence Formula)

\[C_{i,j} = \sum_{s \in S} \text{count}(w_i, w_j \text{ in } s)\]
คำอธิบาย: เรานับว่าคำที่ \(i\) และคำที่ \(j\) ปรากฏร่วมกันในประโยคเดียวกันบ่อยแค่ไหน เพื่อสร้าง "ลายพิมพ์นิ้วมือ" ทางสถิติของแต่ละคำ
📥 Input (ประโยค):
"cat sits mat"
📤 Output (Matrix):
cat & dog จะมีค่าในคอลัมน์ sits สูงเหมือนกัน
⬇️

2. การสลายเมทริกซ์ (SVD Decomposition)

\[A = U \Sigma V^T\]
คำอธิบาย: เราใช้ Singular Value Decomposition เพื่อแยกข้อมูลออกเป็น 3 เมทริกซ์ โดยที่ \(U\) เก็บข้อมูลความสัมพันธ์ระหว่างคำกับ "หัวข้อแฝง" (Latent Topics)
📥 Input:
เมทริกซ์ความถี่ขนาดใหญ่ (Sparse Matrix)
📤 Output:
เมทริกซ์ \(U\) ที่มีมิติเท่ากับจำนวนคำศัพท์
⬇️

3. การเลือกคุณลักษณะสำคัญ (Feature Selection)

\[W_{vector} = U[:, 0:8]\]
คำอธิบาย: เราเลือกเอาเฉพาะ 8 คอลัมน์แรกของเมทริกซ์ \(U\) เพราะเป็นมิติที่มีความแตกต่างของข้อมูล (Variance) สูงที่สุด ซึ่งเพียงพอที่จะใช้เป็น "ความหมาย" ของคำ
📥 Input:
เมทริกซ์ \(U\) ขนาดเต็ม
📤 Output:
เวกเตอร์ขนาด 8 มิติสำหรับแต่ละคำ
⬇️

4. การจัดกลุ่มและการแสดงผล (Clustering & Visualization)

\[\text{dist}(A, B) = \sqrt{\sum (A_i - B_i)^2}\]
คำอธิบาย: เราใช้ Euclidean Distance เพื่อจัดกลุ่มคำที่อยู่ใกล้กัน (Clustering) และใช้เทคนิคพิเศษเพื่อให้กราฟอ่านง่ายขึ้น:
  • Visual Jitter: เพิ่มการกระจายตัวเล็กน้อยเพื่อให้คำที่ซ้อนกันแยกออกจากกันได้
  • Relation Lines: ลากเส้นเชื่อมโยงคำที่อยู่ในกลุ่มเดียวกัน
  • Color Coding: ใช้สีเดียวกันสำหรับคำที่อยู่ใน "ครอบครัว" เดียวกัน
📥 Input:
เวกเตอร์ 8 มิติ
📤 Output:
โครงข่ายความสัมพันธ์ (Visual Semantic Network)
